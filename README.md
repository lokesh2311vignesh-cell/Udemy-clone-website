# Udemy Clone Website

This project is a simple Udemy-inspired website built using HTML and CSS. It replicates the basic layout and design of an online learning platform, including a navigation bar, course categories, featured courses, and responsive sections.

## Features
- Clean and modern user interface
- Navigation menu
- Course listing section
- Categories and featured courses
- Responsive design using CSS
- Beginner-friendly project structure

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to practice front-end web development fundamentals and improve skills in HTML and CSS by recreating the design of a popular e-learning platform.

- Implement course search functionality
- Create login and signup pages
